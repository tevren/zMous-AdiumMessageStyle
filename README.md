# zMous-AdiumMessageStyle

A fork of xMous. This fork adds inline images.

## Prerequisites

Adium v1.4+

## Installation

Clone this repository.
	git clone git://github.com/tevren/zMous-AdiumMessageStyle.git

Double click on zMous.AdiumMessageStyle to install
